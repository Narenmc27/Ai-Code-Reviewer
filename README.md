# 🤖 AI Learning Tools

A 3-in-1 AI-powered platform built with Streamlit and Groq LLM.
Designed to help developers learn faster and work smarter.

## 🔴 Live Demo
https://ai-code-reviewer-jhi7agbezpaa2evdfjyvsr.streamlit.app

## ✨ Features

### 🔍 Code Reviewer
- Paste any Python code
- AI reviews for bugs, improvements and best practices
- Powered by Llama 3.3 via Groq

### 📚 Flashcard Generator  
- Enter any topic
- AI generates 5 study flashcards instantly
- Great for learning new concepts fast

### 📄 Document Q&A (RAG System)
- Upload any PDF document
- Ask questions about it
- AI answers only from your document

## 🛠️ Tech Stack
- **Streamlit** — UI framework
- **Groq + Llama 3.3** — LLM backbone
- **LangChain** — LLM orchestration
- **PyPDF2** — PDF processing
- **RAG System** — Document Q&A

## 🚀 Run Locally
```bash
git clone https://github.com/Narenmc27/ai-learning-tools
cd ai-learning-tools
pip install -r requirements.txt
streamlit run app.py
```

## 🔑 Environment Variables
GROQ_API_KEY=your-groq-api-key
